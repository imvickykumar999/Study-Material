Color Modes-
1)RGB-Every digital screen(Tv,Laptop,Mobile,Desktop ar any other electronic gadget having screen) supports RGB mode. It means red,green and blue pixels, which are from 0-255 = total 256 pixels

2)HSL-Hue, Saturation and Luminance(brightness)
Hue-degree value from color wheel( 0-359 deg)

Saturation-value in percentage(1%-100%), amount of grey color add or subtract.If saturation decrease it will add grey and vice versa.

Luminance(brightness)-Value in percentage(1%-100%), amount of black add or subtract. If luminance decrease it will add black and vice versa.

3)CMYK-Cyan Megenta Yellow Black
Used in publishing(printing)
value in 0-100%
