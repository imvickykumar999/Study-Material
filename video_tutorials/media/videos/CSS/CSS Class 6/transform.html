Types of Transform

 
	Transform an element-
	1)Position Change(translate)
	2)Scale(width/height -Increment/Decrement)-mention in number eg. 1.5 or 2 etc.
	3)Skew-slanted rotate in degree
	4)Rotate-degree or turn
	5)Matrix- combination of multiple task

4. Rotate
